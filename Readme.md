#### DIO – Keylogger & Ransomware (Projeto Educacional - Desafio DIO)

Este repositório contém projetos educacionais desenvolvidos para estudo e compreensão de técnicas utilizadas por malwares, como keyloggers e ransomware.
O objetivo é entender como essas ameaças funcionam, permitindo evoluir conhecimentos em cibersegurança, detecção, prevenção e análise de ameaças.

## Aviso Importante

Este projeto não deve ser usado para fins maliciosos, invasivos ou ilegais.
Todo código apresentado tem finalidade exclusivamente acadêmica, para prática em ambientes controlados e laboratórios pessoais.

## Objetivo do Projeto

Compreender como funcionam keyloggers e ransomwares.

Estudar técnicas de ofuscação, persistência e criptografia usadas por malwares.

Criar um ambiente de laboratório seguro para análise e testes.

Aprender boas práticas de detecção e defesa.

Desenvolver habilidades em Python aplicadas à segurança.
